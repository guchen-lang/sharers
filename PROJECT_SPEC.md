# DevBox

## Project Overview

DevBox is a modern, offline-first developer toolbox built entirely with frontend technologies.

It provides a collection of useful developer utilities inside a single web application.

No backend is required.

All processing must happen inside the browser.

The application should be deployable to Netlify as a static website.

---

# Goals

Create a production-quality developer toolbox that is:

- Fast
- Modern
- Beautiful
- Easy to extend
- Responsive
- Accessible
- Offline-first

The application should feel similar to VSCode or modern IDEs.

# Non Functional Requirements

The application must:

- Work offline
- Support dark mode
- Support mobile devices
- Support keyboard navigation
- Support copy to clipboard
- Remember user input
- Remember settings

# Performance Requirements
